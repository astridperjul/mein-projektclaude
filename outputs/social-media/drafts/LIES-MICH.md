# Drafts – Beiträge zur Kontrolle

Hier legt Claude fertige Beiträge ab, bevor du sie freigibst.

## Ablauf

1. Claude erstellt Beiträge → landen hier in `drafts/`
2. Du liest und prüfst
3. Passt → du verschiebst in `approved/` (oder sagst Claude er soll es tun)
4. Von `approved/` aus wird gepostet (manuell oder via Blotato)

## Dateiformat

Jeder Entwurf bekommt einen Namen wie:
`2026-04-22_instagram_thema.md`

So siehst du auf einen Blick: Datum, Kanal, Thema.
